Detailed Description

Interface definition for the game map abstraction.

This file defines the IGameMap interface, which abstracts the underlying data structure used to store and access game cells. This abstraction allows the game engine to function independently of whether the map is stored as a graph, array, or any other structure.

Updated in Phase P1.2 to conform to SRD v3.2 specification with simplified interface using at() and size() methods.
