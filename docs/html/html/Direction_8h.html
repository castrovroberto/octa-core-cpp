Detailed Description

Direction system for octagonal grid navigation.

This file defines the Direction enum representing the 8 directions in an octagonal grid, along with utility functions for direction manipulation.

The octagonal grid supports 8 directions at 45-degree intervals: N, NE, E, SE, S, SW, W, NW

Rotation functions allow clockwise and counter-clockwise direction changes, which are essential for the chain reaction algorithm in the game logic.

◆ Direction

enum class Direction : uint8_t
strong

Represents the 8 directions in an octagonal grid.

The directions are arranged in clockwise order starting from North. Each direction corresponds to a 45-degree increment around the compass.

The numeric values are intentionally sequential to enable easy rotation calculations using modular arithmetic.
