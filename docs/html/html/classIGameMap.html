Detailed Description

Abstract interface for the game map.

This abstract base class defines the essential operations for a game map, allowing the game engine to function independently of the underlying data structure implementation.

The interface has been simplified in SRD v3.2 to focus on the core operations needed by the game logic:

  • Cell access by coordinate
  • Total cell count for game state queries

Concrete implementations include:

  • GraphGameMap: Hash map-based implementation for sparse grids
  • ArrayGameMap: 2D array-based implementation (future)
